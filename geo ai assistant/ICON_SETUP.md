## Custom Icon

To use a custom icon:
1. Replace `icon.png` in the plugin root
2. Ensure it's 64x64 pixels
3. Use PNG format with transparency
4. Restart QGIS

The icon will appear in:
- QGIS Plugin Manager
- Plugin toolbar
- Plugin menu
